# DonutGlyphChanger

***Using legitimate game copies, purchased through official stores or physical discs, ensures you have clean, unmodified files that work best with Donut Glyph Changer. Original game files provide the most reliable foundation for glyph modifications.***


Donut Glyph Changer
Donut Glyph Changer is a specialized tool designed to replace PlayStation button prompts with Xbox-style ABXY glyphs in compatible PlayStation games. Perfect for players who prefer Xbox controller layouts or are more familiar with ABXY button notation, this Changer seamlessly swaps traditional PlayStation symbols (Cross, Circle, Square, Triangle) with their Xbox equivalents.
Features

Button Glyph Replacement: Convert PlayStation button icons to Xbox ABXY format
Selective Game Support: Works with compatible PlayStation titles that store button graphics in accessible formats

***Use Cases:***

Donut Glyph Changer is ideal for PC ports of PlayStation games where you're using an Xbox controller, emulation setups with Xbox controller input, or simply personal preference for the ABXY button layout. It's particularly useful for players transitioning between console ecosystems who want consistent button prompts across platforms.

***Cross (✕) → A***

***Circle (○) → B***

***Square (□) → X***

***Triangle (△) → Y***

**Important Notes:**
Donut Glyph Changer only works with select games that use compatible file formats for button graphics. Always backup your game files before making modifications. The tool is designed for personal use to enhance your gaming experience with your preferred controller setup.
